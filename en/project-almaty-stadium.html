Almaty International Stadium Project (Detail)

Project type: Sports infrastructure | Status: Planning | Source basis: publicly disclosed information.

Location and Land

The project is planned in Medeu District, Almaty, along Kuldzhinsky Trakt. Public responses indicate land allocation has been made for construction.

Capacity and Positioning

Planned capacity is approximately 45,000 seats, potentially making it one of the largest stadiums in Kazakhstan.

Technical Standard

The concept targets UEFA Category IV requirements with a natural grass pitch and parking for more than 1,000 vehicles.

Timeline

Construction cost is expected to be determined after state review of design and budget documentation. Public target completion is Q4 2027.

Potential Value

The project may improve major event hosting capacity and support surrounding transport, commerce and tourism demand.

Key Watchpoint

A central open point is whether a roofed design will be adopted, which affects all-weather operation and event flexibility.

Update Log

Future updates on budget, schedule, design and functional configuration will be added to this page as disclosed.
